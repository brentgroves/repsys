## 1. Microsoft Fabric Event Stream

- Requires public IP.
- Need to set up port forwarding to the MQTT server.

## 2. KepServerEx

- MQTT driver
- Mirror tags to the database feature.

## 3. Structures ETL system

- Same as KepServerEx
- Create Script

## 4. Python Program

- Subscribe to PLC tags
- Forward tag updates to the database
