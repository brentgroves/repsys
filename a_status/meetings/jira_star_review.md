## Workflow

- create star review space
  - choose work management
  - choose task tracking
- Create separate project space for each project in star review space.
  - Manage tasks to complete star review goal

## Other Atlassian Apps

- **Jira Software:** A project and issue tracking tool designed for agile software development teams.
- **Bitbucket:** A web-based hosting service for projects that use Git version control systems, offering code management and collaboration features.
- **Bamboo:** A continuous integration and continuous deployment server.
- **Jira Work Management:** A project management tool for business teams such as marketing, HR, and finance.
- **Confluence:** A document collaboration and knowledge management tool where teams can create, share, and collaborate on content.
