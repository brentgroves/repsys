# Structures MicroCloud

The Avilla Structures Microcloud will be used the Microsoft Data Gateway. To enable Structures employees to build Power BI dashboards, reports, and notebooks using on-prem data sources such as SQL server and data sources that can not be directly accessed from the Microsoft Fabric environment such as the Plex ERP.

## Components

- LXD Cluster
- OVN Cluster
- Ceph Cluster

## Cost Savings

- 3 Windows Servers is what we save
- Microsoft Azure Blob Storage
