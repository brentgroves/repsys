## Setup

- The Marposs gage computer is powered by a 36V DC converter plugged into a standard 120VAC outlet.
- One of the two Marposs gage computer USB ports is connected to a 10 port USB hub.
- One model B6871250050 Easy Box U8I/O with 8 INPUT/OUTPUT which is not being used.
- Three **[EasyBox U4F-HR, U4H, or U4T transducers](https://www.marposs.com/media/4263/d-1/t-file/D6D02000G0.pdf)** to connect up to 4 MARPOSS standard full-bridge or half-bridge transducers

EthernetIP not modbus or seimen profibus.

## 2.2 Installation (software versions 3.0.xx only)

### 2.2.1 System Requirements

Merlin Plus Software is a Marposs product that can be installed on a PC (embedded, desktop,
laptop or tablet-PC) running Windows® 7, Windows® 10 or Windows® 10 IoT with the following
minimum characteristics:
• Intel® Dual Core D2550 1.86GHz
• 2 GB RAM
• 32 GB Hard Disk
• una porta USB

### Installation Procedure

REMARK:
the Microsoft .NET Framework 3.5 feature must be activated

if this feature is not activated, it is mandatory that the PC can have access to the
Microsoft Update website to complete this operation

- Connect the USB memory pen with the Merlin Plus Software setup
- if the Easy Box U4S has to be used, please install soft-full-U4S_EBCP210X (from the
Drivers folder);
- If the Orbit3 Solartron have to be used, please install Soft-Full-Orbit3-1.3.15 (from the
Drivers folder);
