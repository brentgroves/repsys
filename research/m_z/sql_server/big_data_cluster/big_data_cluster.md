# **[Retirement of SQL Server 2019 Big Data Clusters](https://learn.microsoft.com/en-us/lifecycle/announcements/sql-server-2019-big-data-clusters-retirement)**

SQL Server 2019 Big Data Clusters will be retired on February 28, 2025. Existing users of SQL Server 2019 with Software Assurance will be fully supported on the platform for the next three years, through February 2025. Big Data Clusters will continue to be maintained through SQL Server cumulative updates until that time. Please see the SQL Server blog for more information.

**[SQL Server Big Data Cluster]()**

SQL Server Big Data Clusters (BDC) was a feature introduced in SQL Server 2019 designed to provide a unified data platform for analytics and AI, combining SQL Server with Apache Spark and HDFS. It allowed organizations to manage and analyze large volumes of data, both structured and unstructured, within a single, integrated environment.
