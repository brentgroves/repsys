## The following can leverage the Microsoft On-Premises Data Gateway

- microsoft fabric
- power bi
- pipelines
- data flows
- short cuts
- Azure Analysis Services
- Power Apps, Power Automate
- Azure logic apps

## Requirement

Gateways must be configured in the Linamar Microsoft Tenant.
