# **[show db](https://learn.microsoft.com/en-us/cli/azure/sql/db?view=azure-cli-latest#az-sql-db-show)**

```bash
az sql db show [--expand-keys {false, true}]
               [--ids]
               [--keys-filter]
               [--name]
               [--resource-group]
               [--server]
               [--subscription]
```
